# AngularVideoPlayer

The Angualr project for Video Player

Git: https://github.com/keenkit/FullStack/tree/master/AngularVideoPlayer

# How to run?

```
npm install

# Start the json-server:
json-server --watch db.json

# Start the app
ng serve

# Open the browser http://localhost:4200
```
